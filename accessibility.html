Accessibility

We have tried our best to maximize the accessibility.

  1. All non-text elements have a text alternative so that screen readers should have no problems with describing all non-text elements of the page.
  2. All functionality of our website are available from keyboard.
  3. All of the elements of this website are performed in good contrast.
  4. There is a sitemap page avaiable which hopefully can let user find things easier.
